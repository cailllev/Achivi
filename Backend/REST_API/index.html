Achivi REST API

Example API

delete
/api/example/{example_id}
Deletes an example
get
/api/example/{example_id}
Retrieves an example
post
/api/example
Creates and saves an example

Implementations Notes

  • exp_id is an identifier generated by the DB
  • exp_number is the number in the example
  • exp_text is the text in the example
  • exp_enum choose between 'winner' and 'spinner'
  • exp_created_at and exp_updated_at are fields which are generated by the back-end
  • client_id is an identifier that a client should generate (could be used for online/offline sync)
